API 文档

提示：所有 API 端点都以 /twitter/api/ 为基础路径

监控用户 API

GET /twitter/api/monitored-users/
获取所有监控用户列表（支持分页、搜索、过滤）
示例请求：
curl http://localhost:8000/twitter/api/monitored-users/
POST /twitter/api/monitored-users/add_user/
添加新的监控用户
示例请求：
curl -X POST http://localhost:8000/twitter/api/monitored-users/add_user/ \
  -H "Content-Type: application/json" \
  -d '{"username": "elonmusk"}'
POST /twitter/api/monitored-users/{id}/monitor_now/
立即监控指定用户
示例请求：
curl -X POST http://localhost:8000/twitter/api/monitored-users/1/monitor_now/
POST /twitter/api/monitored-users/monitor_all/
监控所有启用的用户
示例请求：
curl -X POST http://localhost:8000/twitter/api/monitored-users/monitor_all/

推文 API

GET /twitter/api/tweets/
获取所有推文（支持分页、搜索、过滤、排序）
示例请求：
curl http://localhost:8000/twitter/api/tweets/?author=1&ordering=-created_at
GET /twitter/api/tweets/{id}/replies/
获取指定推文的所有回复
示例请求：
curl http://localhost:8000/twitter/api/tweets/1/replies/

回复 API

GET /twitter/api/replies/
获取所有回复（支持分页、搜索、过滤）
示例请求：
curl http://localhost:8000/twitter/api/replies/?author=1

监控日志 API

GET /twitter/api/logs/
获取所有监控日志
示例请求：
curl http://localhost:8000/twitter/api/logs/?user=1&status=success
